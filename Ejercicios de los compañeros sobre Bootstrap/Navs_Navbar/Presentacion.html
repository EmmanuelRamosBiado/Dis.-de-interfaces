¿Qué es el nav?

Los navs son una barra de navegación que se puede utilizar para agrupar enlaces de navegación, formularios o botones. Se utilizan para crear barras de navegación, barras de pestañas y otros elementos de navegación. Son flexibles y se pueden adaptar a cualquier tipo de diseño, ya sea horizontal o vertical. Pueden utilizarse con la clase .nav o con la clase .navbar para crear barras de navegación.

Los navs se pueden alinear a la izquierda, al centro o a la derecha con la clase .justify-content-start, .justify-content-center o .justify-content-end. También podemos utilizar la clase .nav-fill para que los elementos de la barra de navegación ocupen todo el ancho disponible.

¿Qué es el navbar?

El navbar es una barra de navegación que se encuentra en la parte superior de la página web, y que sirve para que el usuario pueda navegar por las diferentes páginas de la web. En bootstrap, el navbar se puede crear de dos formas: con la clase .navbar o con la clase .navbar-expand.
